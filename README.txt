Aplikasi Data Mining berbasis Streamlit
Files:
- streamlit_data_mining_app.py : aplikasi utama
- requirements.txt : dependensi

Jalankan:
1. (opsional) python -m venv venv
2. source venv/bin/activate  (atau venv\Scripts\activate di Windows)
3. pip install -r requirements.txt
4. streamlit run streamlit_data_mining_app.py
